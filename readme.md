## Install dependencies
```
npm install
```

## Input Github's api Token (Github Profile Settings-->Personal Access Tokens)
```
./client/services/github.service.ts
secretToken:string= "<GITHUB TOKEN HERE>";
```


## start server
```
node ./bin/www
```
